Getting started

Signkick CSS framework

This CSS framework build for Signkick need and uses its styleguides. It has a modified version of Bootstrap in its core codebase and uses BEM methodology.

Architecture

Global frontend architecture is a combination of Angular 2 principals with its component-centured approach, and BEM methodology to make code DRY, improve the web performance and standardise visual elements.

Front end source code is distributed in 3 directories:

src/

├─ app/ - custom Angular components

├─ base/ - CSS framework

├─ theme/ - css variables sorted by type

base/ directory includes global css settings, normalise and reboot files, styles of basic reusable UI elements, utilities and mixings

base/

├─ _base.scss - compounding all files

├─ _mixins.scss - compounding all mixings

├─ _mixins/

GLOBAL STYLES
