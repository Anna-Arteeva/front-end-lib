Grid

There is no fixed column based system (standart for Bootstrap) set here. Instead I use combination of flex utilities that are more flexible and customisable. Plus, much less code.

Standard grid automatically adjust to the number of children blocks and make each of equal width. There are 2 variations of standard grid:

1. Grid without indent between blocks
<div class="grid-{breackpoint}">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

Custom column width

If you would like to make colomns with non-equal widths, you can use on of the pre-made classes with custom flex-basis number, or create a new one.

Persentage (responsive)

Fixed column width with number in percent, you can specify the width just for some columns, then the others take the rest of the space with equal width each. The width is valid only before the parent grid breakpoint, on smaller resolutions it will take 100% of the space. There are 4 pre-made responsive classes:

grid-col-10per-{breackpoint}
grid-col-30per-{breackpoint}
grid-col-50per-{breackpoint}
grid-col-70per-{breackpoint}

Fixed width

You can create a column with fixed width set in pixels, rem or em. Same like in per cent option: you can specify the width just for some columns, then the others take the rest of the space with equal width each. There are 4 pre-made responsive classes:

grid-col-20rem
grid-col-30rem
grid-col-50rem
grid-col-100rem

Create custom column

Letâ€™s create a responsive grid with 4 columns, one custom and 3 equal sized.

1. The grid would break on small devises, so each column would take 100% of the available width. Therefore we can use default class for that grid-sm.

<div class="grid-sm">
    <div class="grid-col__custom">Custom column</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

2. On large devices the first column would take 50% of the space, but not less then 500px. So we make a custom class .grid-col__custom with breakpoint on lg. Add flex-grow: 0; to forbid the column to take more space, then 50%

.grid-col__custom {

    @include media-breakpoint-up(lg) {
        flex-basis: 50%;
        flex-grow: 0;
        min-width: 500px;
    }
}

3. On medium devices the first column would take 100% of the space, while other columns move to the next row and take the equal widths there. For that we add flex-basis: 100%; to the .grid-col__custom.

.grid-col__custom {
    flex-basis: 100%;
    @include media-breakpoint-up(lg) {
        flex-basis: 50%;
        flex-grow: 0;
        min-width: 500px;
    }
}

4. But because wrapping is not allowed for the parent grid by default, we add a modifier class to change it on medium devices

.grid__custom {
    @include media-breakpoint-down(md) {
        flex-wrap: wrap;
    }
}

So html should look like that

<div class="grid-sm">
    <div class="grid-col__custom">Custom column</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>
