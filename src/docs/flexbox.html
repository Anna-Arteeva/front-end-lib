Flexbox utilities

Flexbox is a great and flexible way to organise set of blocks and build complex responsive layouts. Find here easy quick-to-use utilities for the most common cases.

Find out more about flexbox in this detailed flexbox guide

Responsive flexbox shortcut
class: flex (all dimentions)
responsive: flex-xs, flex-sm, flex-md, flex-lg, flex-xl
Example

flexbox for medium (md) and bigger screen size flex-md (resize the browser window to see the difference)

1
2
3
4
Example code
<div class="flex-md">
    <div>1</div>
    <div>2</div>
    <div class="display-1">3</div>
    <div>4</div>
</div>
